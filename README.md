# React Query Graphql

A project to learn React Query

## Installation

```bash
yarn
```

## Setup

Create a `.env` file with:

```
NEXT_PUBLIC_API_URL=https://rickandmortyapi.com/graphql
```

## Technologies used

- React Query
- GraphQL
- Apollo CLI (codegen)
- TypeScript
- Next.JS
